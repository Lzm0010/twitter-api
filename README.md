# Build Twitter Interface

### Installation

```
$ npm install

```
add a config.js file to main directory with code modeled like example at very bottom

```
$ npm start
```
### Please add your twitter access keys to the config.js file you created
```
module.exports = {
      consumer_key: ''
    , consumer_secret: ''
    , access_token: ''
    , access_token_secret: ''
};
```
